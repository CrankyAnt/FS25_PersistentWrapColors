# Distribution and Branding Policy

## Per-File Licensing

This repository uses the REUSE specification to assign licenses per file.
`REUSE.toml` is the authoritative machine-readable record of which license
applies to each file.

The functional Lua source code and supporting technical repository files are
licensed under the MIT License. You may fork, modify, and distribute those
files under the terms of that license.

The private `docs` Git submodule is a separate repository and is not covered
by the licensing declarations in this repository.

## Branding and Official Releases

Files identified in `REUSE.toml` as
`LicenseRef-CrankyAnt-Official-Assets` are covered by the
[CrankyAnt Official Assets License](LICENSES/LicenseRef-CrankyAnt-Official-Assets.txt)
instead of the MIT License. These include:

- The name "Persistent Wrap Colors"
- The official mod icon and other branding assets
- The official ModHub description and promotional materials
- Official release packages produced by CrankyAnt

Official, unmodified releases are distributed only through:

- GitHub: https://github.com/CrankyAnt/FS25_PersistentWrapColors
- GIANTS ModHub: https://www.farming-simulator.com/mod.php?mod_id=334607&title=fs2025

Automated scraping, mirroring, or rehosting of official release packages,
branding assets, or promotional materials is not permitted.

## Modified Versions

Modified versions of the MIT-licensed files may be distributed under the
terms of that license. When distributed as a complete unofficial mod using
adapted covered configuration or documentation, the modified version must:

- Use a different mod name and icon
- Clearly identify itself as an unofficial modification
- Preserve applicable copyright and license notices
- Not imply endorsement by or affiliation with CrankyAnt

Covered configuration and documentation files may be adapted for such an
unofficial modified version under the terms of the CrankyAnt Official Assets
License.

Linking to the official GitHub repository or GIANTS ModHub listing is
permitted and encouraged.
